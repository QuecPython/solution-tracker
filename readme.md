## Tracker介绍

### Tracker

- Tracker, 智能跟踪定位器。是专门用于监控宿主的位置信息与运动状态, 以及对设备本身信息, 收集与整理后, 将设备信息与宿主信息通过无线网络传送至云端服务器的无线终端设备。
- 业务逻辑:
    - 在云端设置好需要采集的数据信息, 报警信息和控制参数;
    - 传感器或内置模块采集对应的数据信息, Quecpython整合好数据后, 通过MQTT协议发送至云端, 云端接收到数据可以进行分析、处理、显示、保存等操作;
    - 手机端向云端发送控制指令, 云端下发指令到设备模块, 控制模块的数据采集与上报的启用, 禁用和采集周期等。
